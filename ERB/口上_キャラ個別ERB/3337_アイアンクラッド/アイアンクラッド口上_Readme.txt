;●バトルコンセプト
;--------------------------------------------------
;・スレスパの再現として、「山札とドロー」システムを再現
;・使用できるアビリティがターンごとに変化するぶん、攻撃性能は高め
;・その他、強力なバリア、永続の自己強化アビリティを有する
;・ただし、特に強力なアビリティは消耗がひどく激しい。MPも低いため継戦能力はいまいち
;・実質の外れアビとして防御系も多めに

;●メモ
;--------------------------------------------------
;・戦闘コンセプト
;　スレスパ系キャラで共通したシステムを作りたくて……
;　ほかにキャラを作るかは不明なり。
;　というか調整が大変すぎる……
;・一人称、二人称
;　それぞれ、「俺」「お前」としていますが、実際のところ不明です。
;　「早くしろ」と他人に言えるし、パーティプレイを楽しんだり、程度の意識はあるようですが、敵を前にしたら「敵、殺す……」くらいしか言えないっぽいしねえ……
