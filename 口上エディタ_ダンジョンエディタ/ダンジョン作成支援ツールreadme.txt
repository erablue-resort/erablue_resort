
****************************************
ダンジョン作成支援ツールreadme
****************************************

・概要
	erablue_resort用のダンジョン作成支援ツールです。
	ダンジョンマスの中身をメモりながら、マスとマスの接続や表示するアイコンなどを設定出来ます。
	とはいえERBの記述をこれでやることは想定していません。
	骨格だけを作り、細部の修正やイベントの処理などはちゃんとしたテキストエディタで行うことを想定しています。

・使い方
	起動した後、ダンジョン名と床画像名を入力する。

	任意のマスを選択しセルイベントの内部にイベントメモを記入する。
	その後、「接続先を指定する」のボタンを押してマスとマスの繋がりを設定する。
	（設定した接続先はボタンの横に表示される）

	必要な分マスへの記入を繰り返してダンジョンの骨格を作る。以上。
	

	また、ERBを読み込む機能は搭載していません。
	「読み込み」「保存」ボタンではCSVファイルに一旦データを保存する形式になっています。
	「ERB出力」はツールでの設定が全て終わった後に実行してください。

	セル画像部分に並べてあるボタンは、マスに重ねて表示するアイコンの設定ボタンです。
	それぞれ必要な時に設定してください。

	セルイベントの右に並べてあるボタンは、イベント関数のテンプレをセルイベントに突っ込むボタンです。
	ERBを出力した後にイベント処理を記入する手間がちょっとだけ省かれます。

・ライセンス
	改造でも再配布でも好きにしてOKです。
