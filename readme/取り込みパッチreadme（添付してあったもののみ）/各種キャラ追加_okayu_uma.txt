・ウーフ＆レニー
ウーフは「紳士的な怪物」でレニーは「年不相応に賢いが年相応に幼い少女」
ウーフ
レニーに対する対応、敵対相手に対して積極的な反撃をするわけでもない
反発して離れたレニーに対してもちゃんと心配する、等外見以外は普通の大人
銃・格闘が得意…となっているが、イラスト的には斧っぽい印象

レニー
年齢より少し賢いせいで周りに溶け込めなかった少女
ウーフに対して脅しをかける度胸、ウーフに庇護されつつもそのウーフに対する当たりの強さ（試し行為ではあると思うが）
ウーフ以外の他人との対応とか子供らしい我侭さを含めてつけてみたら並んだ素質がひどいことひどいこと…
