画像制作者ではありませんが(ry
ネームエラーのリネームをしたときに顔画像の見切れや構図ミスっぽいものが気になったので短冊から顔画像を切り取り加工してみました。
基本的には頭がメインにきていて顔が見切れているものを顔または奉仕部位メインに加工しています。
これはあくまで私の感性ではこっちの方がシコくない？といった一案であり、画像制作者様の皆様にはいつも素晴らしい画像を作成していただき感謝感激しております。
