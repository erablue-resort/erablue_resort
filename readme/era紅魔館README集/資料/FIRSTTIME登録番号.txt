FIRSTTIME関数に用いる番号表です。
IF FIRSTTIME(20)
	PRINTFORML 初キス口上
ELSE
…
のような使い方をします。
この場合ではFIRSTTIMEに20という引数(文字列も可)を送り、CSTR:0に登録されている文字列に当てはまるものがあれば既に経験済みとして0を返す、
つまりIF FIRSTTIME(20)は偽となり初キス口上は表示されません。
もしCSTR:0に当てはまる文字列がなければCSTRに渡された文字列(この場合は20)を登録して1を返します。
つまりIF FIRSTTIME(20)は真となり初キス口上が表示されます。

省略しない形ではFIRSTTIME(文字列,引数1,引数2)という形になっており、
文字列はCSTR:0に登録、または検索するために用います。

引数1を真（0以外）にするとでCSTR:0に文字列を登録するか否かを設定出来ます。
たとえば口上の表示前に
IF FIRSTTIME(20,1)
	PRINTFORM キス未経験の口上
のような使い方ができます。このケースではキス未経験かどうかの判別だけを行っています。
もし
IF FIRSTTIME(20)
	PRINTFORM キス未経験の口上
とするとキス調教が登録され、以後のFIRSTTIMEに反映されてしまいますが、それを避けることが出来ます。

引数2でCSTR:0のキャラ番号を設定出来ます。口上での使用を楽にするため省略、もしくは0にした場合はTARGETとなります。
MASTER（キャラ番号0）を参照したい場合は-1を代入してください。

0～500 SELECTCOMに対応
500～ 派生コマンド

精飲 初めてPLAYERの精液を口に入れた
射精0～ TFLAG:1のビットに対応
