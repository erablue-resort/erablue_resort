見た目上の変化
・好感度の高い住人が少し%CALLNAME:MASTER%に寄って来やすく
・見通せる隣接箇所の住人の動きが分かる(正門←→庭や、庭←二階の部屋など)
・見通せる隣接箇所の住人に情事がバレる
　# 特に庭は、こちらからは見えない二階の部屋から見つかる恐れ有り

内部的な変化
・@CAN_MOVEを式中関数化
　# CAN_MOVE(現在の場所, 行き先)という使い方になります
　　行き来が可能なら1が返ります
　　行き来が可能で見通せる場所なら2が返ります
・@AFFAIR_DISCLOSUREにARG:1追加
　# ARG:1が0なら今まで通り
　　1だと覗き状態なので口上は出ませんが情事は発覚します
　　どうして%CALLNAME:MASTER%が覗かれたことに気づいたのかは想像にお任せします
