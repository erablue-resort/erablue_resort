◇このパッチの内容◇
軽食の他に主食とデザートを作れるようになるパッチ。
そして軽食なら食堂、もしくは居間以外でも食べられ、何回でも作れます。ただし回復量はパッチ当てる前よりも低めです。
[食事をふるまう]コマンド追加
	作った料理を自分で食べずに誰かに食べさせられ、部屋にお届けも可能。
	また主人公以外にも食事フラグを使い、軽食以外は一日一回のみ。
*NEW*
	時間経過で作った料理が冷めたり腐ったりします。
	それでも一応食べさせる事ができますが、余りにも時間が経っていると反感を買います。
*NEW*
	料理に何か薬を混ぜられます。が、現状効果はなし。次回でなにかしら実装予定。
